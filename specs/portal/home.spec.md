# 门户首页 Spec

## 概述

AI 辅助研发平台的门户首页，聚合展示各模块的精华内容，起到导航和吸引访问的作用。

---

## 页面布局

```
[Hero Banner]
[最新公告（3条）]
[精选课程（6张卡片）]
[能力市场亮点（4张卡片）]
[优秀案例集锦（3张卡片）]
[荣誉墙入口（最新3条）]
```

---

## 接口依赖

| 区块 | 接口 | 参数 |
|------|------|------|
| 最新公告 | GET /api/announcements | page=1, pageSize=3 |
| 精选课程 | GET /api/courses | page=1, pageSize=6 |
| 能力亮点 | GET /api/capabilities | page=1, pageSize=4 |
| 优秀案例 | GET /api/cases | page=1, pageSize=3 |
| 荣誉墙 | GET /api/honors | page=1, pageSize=3 |

---

## 页面行为

- **并行请求**：五个区块数据并行拉取，各自独立展示加载/成功/错误状态
- **Hero Banner**：静态内容（标题 + 副标题 + CTA 按钮），不依赖接口
- 各区块底部有「查看全部」链接，跳转对应列表页
- 某区块接口失败时，该区块显示「加载失败」，不影响其他区块

---

## 验收标准（Acceptance Criteria）

- [ ] 五个区块数据并行加载，任一区块失败不影响其他
- [ ] Hero Banner 无闪烁，首屏可见
- [ ] 所有「查看全部」链接跳转正确
- [ ] 首页在 3G 网络下 LCP（最大内容绘制）不超过 3 秒
